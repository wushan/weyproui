Trigger

(automatic) you don't neeed trigger this with the classes mentioned above.

$('.data-tables.normal').DataTable({
	paging: true,
	"pagingType": "full",
    rowReorder: {
		selector: 'td:first-child',
		dataSrc: 'td:first-child'
	},
	ordering: true
});
