Structure

Add class : 'data-tables','normal'

table.rowed.table-hover.list-table.data-tables.normal
	thead
		tr.centered
			th(width='5%') A
			th(width='5%') B	
			th(width='6%') C
			th(width='40%') D
			th(width='15%') E
			th(width='17%') F
	tbody
		each i in Array(10)
			tr
				td.centered AA
				td.centered BB
				td.centered CC
				td
					+lipsum(1,'sentences')
				td.centered EE
				td.centered
					a.btn.basic(href='javascript:;') 編輯
					a.btn.delete(href='javascript:;') 刪除

Trigger

(automatic) you don't neeed trigger this with the classes mentioned above.

$('.data-tables.normal').DataTable({
	paging: true,
	"pagingType": "full",
    rowReorder: {
		selector: 'td:first-child',
		dataSrc: 'td:first-child'
	},
	ordering: true
});
